# dbt project (Favorita)

dbt models, tests, macros, and profiles for the Favorita forecasting pipeline. All commands run in Docker via the repo root Makefile (see [../README.md](../README.md) for setup, credentials, and CI).

## Layout

```
dbt/
  docs/           # Project overview for dbt Docs (overview.md)
  models/
    staging/      # Staging models
    intermediate/ # Feature sets (int_sales_*)
    marts/        # Marts and BQML outputs
      ml_models/  # BigQuery ML models
  macros/         # BigQuery ML helpers
  profiles/       # Connection profiles (mounted in Docker)
  selectors.yml   # daily_refresh and related selectors
  tests/          # Singular data tests
```

## Common commands (from repository root)

```bash
make dbt-debug
make dbt-deps
make dbt-run
make selector-daily-refresh
make selector-daily-refresh-test
make dbt-train          # BQML train tag
make dbt-predict        # BQML predict tag
make dbt-vertex         # Staging over Vertex output tables
make dbt-test
make dbt-docs-generate
```

Pass extra dbt CLI flags with `ARGS`, for example:

```bash
make dbt-run-model MODEL=int_sales_daily
make dbt-test ARGS="--select assert_sales_fct_row_count"
```
